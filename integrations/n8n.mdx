n8n is a powerful workflow automation tool that can be enhanced with AI capabilities through Emby. This guide shows how to integrate Emby into your n8n workflows.

## Prerequisites

- An Emby account with an API key
- n8n instance (self-hosted or cloud)
- Basic understanding of n8n workflows

## Setup

The easiest way to use Emby with n8n is through the OpenAI node with custom configuration.

<Steps>
<Step>
### Add OpenAI Credentials

1. In n8n, go to **Settings** → **Credentials**

2. Click **Add Credential** → **OpenAI**

3. Configure as follows:
   - **API Key**: Your Emby API key
   - **Base URL**: `https://dev.emby.ai/v1`
   - **Organization ID**: Leave blank

</Step>
<Step>
### Configure OpenAI Node

1. Add an **AI Agent** node to your workflow
2. Add a **Chat Model** edge to the node

3. Configure the node to use the Emby provider

<Callout type="warning">
	Note: You have to toggle off the responses API. Emby does not support
	it.
</Callout>

    4. Select your desired options

- **Model**: Use any [Emby model](https://dev.emby.ai/models) ID (e.g., `gpt-5`)
- **Options**: Optionally, configure LLM parameters

</Step>

<Step>
### Test Workflow

Finally, try running your workflow with a test prompt.

</Step>
</Steps>
